# Hava Durumu Takip Sistemi (Bulut Bilişim Projesi)

Bu proje, AWS bulut altyapısı üzerinde çalışan, React ve Node.js kullanılarak geliştirilmiş bir web uygulamasıdır.

##  Proje Özellikleri ve Yapılanlar
* **Bulut Altyapısı:** AWS EC2 Instance (Ubuntu 24.04) üzerinde barındırılmaktadır.
* **Sunucu Yönetimi:** Uygulama kesintisiz çalışması için **PM2** process manager ile yönetilmektedir.
* **Güvenlik:** AWS Security Groups üzerinden 3000 portu (Inbound Rules) tüm dünyaya açılmıştır.
* **Frontend:** React.js kullanılarak kullanıcı dostu, responsive bir arayüz geliştirilmiştir.
* **Backend:** Node.js ve Express frameworkü ile hava durumu API entegrasyonu sağlanmıştır.
